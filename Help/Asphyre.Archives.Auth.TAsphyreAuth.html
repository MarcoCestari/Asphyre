Description

Methods

function Authorize(Sender: TObject; Archive: TAsphyreArchive): Boolean;

Provides password authentication to the provided Asphyre archive by calling existing callback functions to provide the password. If the password has been provided, the returned value is True and False otherwise.

function Subscribe(Callback: TAuthCallback): Cardinal;

Subscribes a new callback function that will provide secure password authentication to Asphyre archives. The returned ID can be used to unsubscribe the callback function.

procedure ProvideKey(Key: Pointer);

Provides the password to the archive that is currently being authenticated. The Key parameter should point to a valid memory block of 16 bytes that contain 128-bit password.

procedure Unauthorize();

Removes and burns the provided password for the existing Asphyre archive so that the password cannot be acquired by a third party.

procedure Unsubscribe(EventID: Cardinal);

Removes the subscribed callback function from the list.
