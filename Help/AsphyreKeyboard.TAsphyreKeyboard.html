Description

Methods

function Initialize(): Boolean;

Initializes the component and prepares DirectInput interface. If the method succeeds, the returned value is True and False otherwise. WindowHandle must be properly set before for this call to succeed.

function Update(): Boolean;

Updates the state of keyboard keys. The returned value is True if the method succeeds and False otherwise.

procedure Finalize();

Finalizes the component releasing DirectInput interface.

Properties

property Background: Boolean read FBackground write FBackground;

Determines whether the input should still be available when application is minimized or not focused.

property Initialized: Boolean read FInitialized;

Indicates whether the component has been properly initialized and is ready to be used.

property Key[KeyNum: Integer]: Boolean read GetKey;

Returns the status of a single keyboard key. KeyNum is the internal key number as defined in DirectInput. These constants are usually named as DIK_[KeyName]. Search for Microsoft article called "Keyboard Device Enumeration" for more information.

property KeyName[KeyNum: Integer]: StdString read GetKeyName;

Returns the name of the given key as it is described by underlying OS. KeyNum is the internal key number as with Key property.

property KeyPressed[KeyNum: Integer]: Boolean read GetKeyPressed;

Indicates whether the key has been pressed after two consequent calls to Update. KeyNum is the internal key number as with Key property.

property KeyReleased[KeyNum: Integer]: Boolean read GetKeyReleased;

Indicates whether the key has been released after two consequent calls to Update. KeyNum is the internal key number as with Key property.

property VKey[VCode: Cardinal]: Boolean read GetVKey;

Returns the status of a single keyboard key. VCode is the virtual key code usually defined by VK_[KeyName] constants.

property VKeyName[VCode: Cardinal]: StdString read GetVKeyName;

Returns the name of the given key as it is described by underlying OS. VCode is the virtual key code.

property VKeyPressed[VCode: Cardinal]: Boolean read GetVKeyPressed;

Indicates whether the key has been pressed after two consequent calls to Update. VCode is the virtual key code.

property VKeyReleased[VCode: Cardinal]: Boolean read GetVKeyReleased;

Indicates whether the key has been released after two consequent calls to Update. VCode is the virtual key code.

property WindowHandle: THandle read FWindowHandle write FWindowHandle;

The handle of the application's main window. This should be properly set before initializing the component as it will not work otherwise.
