Functions and Procedures

function Base64Binary(const Source: AnsiString; Dest: Pointer): Integer;

Decodes Base-64 encoded text back into binary form and copies it to the destination buffer, returning the number of bytes actually written.

function Base64String(Source: Pointer; Size: Integer): AnsiString;

Encodes the specified binary buffer into Base-64 readable text and returns the resulting string.

procedure CipherDataXTEA(Source, Dest: Pointer; Count: Integer; const Key: TKey128; const InitVec: TBlock64);

Encrypts data using 128-bit XTEA cipher in CBC chaining mode and residual block termination (in case the data buffer is not multiple of 8 bytes).

function CompressData(Source, Dest: Pointer; SourceSize, MaxDestSize: Cardinal; Compression: TCompressionLevel = clNormal): Integer;

Compresses data block from the source buffer to destination buffer. The memory for both buffers must be allocated. MaxDestSize can be used to specify the maximum data length that can be saved at the destination buffer to prevent overflow.

function ComputeCRC32(Source: Pointer; Size: Integer): LongWord;

Calculates CRC32 checksum from the specified data. This checkum can be used in integrity tests to verify that the data has not been modified.

procedure DecipherDataXTEA(Source, Dest: Pointer; Count: Integer; const Key: TKey128; const InitVec: TBlock64);

Decrypts data using 128-bit XTEA cipher in CBC chaining mode and residual block termination (in case the data buffer is not multiple of 8 bytes).

function DecodeBase64(Source, Dest: Pointer; Size: Integer): Integer;

Decodes the Base-64 readable text back into the binary format.

function DecompressData(Source, Dest: Pointer; SourceSize, MaxDestSize: Cardinal): Integer;

Decompresses data block from the source buffer to destination buffer. The memory for both buffers must be allocated. MaxDestSize can be used to specify the maximum data length that can be saved at the destination buffer to prevent overflow.

function EncodeBase64(Source, Dest: Pointer; Size: Integer): Integer;

Encodes the binary source into Base-64 readable text. This effectively increases the size by 4/3 (rounded up). If the source is not divisible by 3, it will be padded with zeros.

procedure MD5Checksum(Source: Pointer; Size: Integer; Checksum: Pointer);

Calculates and returns 128-bit MD5 checksum of the given data block. The resulting checksum occupies 16 bytes of memory, which should be previously allocated. This checksum can be used as a signature to verify that the data has not been altered in any way.

TBlock64 = packed array[0..1] of LongWord;

64-bit data block commonly used as init-vector for XTEA cipher.

TCompressionLevel = (...);

The level of compression used when compressing data blocks.

Values
  • clLowest: Lowest level of compression but fastest performance.
  • clNormal: A balanced approach using average compression and performance.
  • clHighest: The best possible compression at the expense of decreased performance.
TKey128 = packed array[0..3] of LongWord;

128-bit data block commonly used as password XTEA cipher.
