Description

Types

TAsphyreTimingMode = (...);

The timing mode that is currently being used by TAsphyreTiming. It is typically chosen the first time one of the routines is called and remains unchanged during the lifetime of application.

Values
  • atmNotStarted: The timing operations have not yet started. One of the timing functions need to be called for this state to change.

  • atmLowPrecision: The timing operations are running in low-precision mode that may degrade the quality of operations. This means that the system has no high precision timer available.

  • atmHighPrecision: The timing operations are running in high-precision mode providing the best possible timer resolution. The actual resolution is hardware and platform dependent and can vary on different configurations.

Variables

Timing: TAsphyreTiming;

A running instance of TAsphyreTiming that is created when the application is executed and freed upon termination; therefore, this class can be used with its timing functions without having to explicitly create it elsewhere.
