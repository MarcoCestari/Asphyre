Functions and Procedures

function Angle3(const a, b: TVector3): Single;

Calculates the angle between two 3D vectors. The returning value has range of [0..Pi].

function ColorToVec3(Color: Cardinal): TVector3;

Converts Red, Green and Blue components of 32-bit color to X, Y and Z components of 3D vector accordingly. This can be useful for making calculations with the color as if it was a 3D vector.

function Cross3(const a, b: TVector3): TVector3;

Calculates the cross-product of the specified 3D vectors. The resulting vector is perpendicular to both source vectors and normal to the plane containing them.

function Dot3(const a, b: TVector3): Single;

Calculates the dot-product of the specified 3D vectors. The dot-product is an indirect measure of the angle between two vectors.

function Length3(const v: TVector3): Single;

Returns the length of the specified 3D vector.

function Lerp3(const v0, v1: TVector3; Alpha: Single): TVector3;

Interpolates between the specified two 3D vectors.

Parameters
v0
The first vector to be used in the interpolation
v1
The second vector to be used in the interpolation
Alpha
The mixture of the two vectors with the a range of [0..1].
function Norm3(const v: TVector3; ZeroAmpEpsilon: Single = 0.00001): TVector3;

Normalizes the specified 3D vector to unity length. The second parameter is used to prevent division by zero in vectors that are of zero length.

function Parallel3(const v, n: TVector3): TVector3;

Calculates the portion of 3D vector "v" that is parallel to the vector "n".

function Perp3(const v, n: TVector3): TVector3;

Calculates the portion of 3D vector "v" that is perpendicular to the vector "n".

function Reflect3(const i, n: TVector3): TVector3;

Calculates 3D vector that is a reflection of vector "i" from surface given by the normal "n".

function SameVec3(const a, b: TVector3; Epsilon: Single = 0.0001): Boolean;

Compares the two specified 3D vectors using the given threshold.

function Vec3ToColor(const v: TVector3): Cardinal;

Converts X, Y and Z components of 3D vector to Red, Green and Blue components of 32-bit color accordingly. The resulting color has Alpha value of 255.

function Vec3toString(const v: TVector3): string;

Returns a string that describes the source 3D vector with two decimal places precision; for example "(0.25, 0.31, 1.45)".

function Vector3(const p: TPoint2; z: Single): TVector3; overload;

Creates a TVector3 record using 2D vector and specified Z coordinate.

function Vector3(x, y, z: Single): TVector3; overload;

Creates a TVector3 record using the specified coordinates.
