Description

Methods

function GetResourceView(): Pointer; virtual;

Returns the pointer to shader resource view when used in latest DX10+ providers.

function Initialize(): Boolean;

Initializes the texture creating all provider specific resources. If this method succeeds, the texture can be used for rendering and the returned value is True. If the returned value is False, it means that the texture initialization failed and configuration parameters need to be revised.

function LogicalToPixel(const Pos: TPoint2): TPoint2px;

Converts 2D logic texture coordinates in range of [0..1] to pixel coordinates.

function PixelToLogical(const Pos: TPoint2px): TPoint2; overload;

Converts 2D integer pixel coordinates to their logical representation provided in range of [0..1].

function PixelToLogical(const Pos: TPoint2): TPoint2; overload;

Converts 2D floating-point pixel coordinates to their logical representation provided in range of [0..1].

procedure Bind(Stage: Integer); virtual;

Binds the texture to the given stage index in DX9- and OGL providers.

procedure Finalize();

Finalizes the texture releasing all provider specific resources.

procedure UpdateMipmaps(); virtual;

Updates all mipmap images contained in the texture. This should only be used when Mipmapping is set to True.

Properties

property Active: Boolean read FActive;

Indicates whether the texture has been created and initialized properly.

property BytesPerPixel: Integer read GetBytesPerPixel;

Indicates how many bytes each pixel in texture occupies.

property Format: TAsphyrePixelFormat read FFormat write SetFormat;

The pixel format of the texture's surface and its mipmap levels.

property Height: Integer index 1 read FHeight write SetSize;

The height of texture's surface or first mipmap level.

property Mipmapping: Boolean read FMipMapping write SetMipmapping;

Determines whether mipmapping should be used for this texture. If this parameter is set to True, a full set of mipmap levels will be used in the texture and handled by its specific provider implementation. Mipmapping is used when the texture is drawn in significantly smaller sizes to improve visual quality of the displayed image.

property Width : Integer index 0 read FWidth write SetSize;

The width of texture's surface or first mipmap level.
