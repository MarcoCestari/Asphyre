Description

Methods

function Initialize(): Boolean;

Initializes the component and prepares DirectInput interface. If the method succeeds, the returned value is True and False otherwise. WindowHandle must be properly set before for this call to succeed. This also updates the number of joysticks present in the system and initializes each one of them.

function Update(): Boolean;

Updates the status of all joysticks in the system.

procedure Finalize();

Finalizes the component releasing all previously created joysticks and the related DirectInput interfaces.

Properties

property Background: Boolean read FBackground write FBackground;

Determines whether the input should still be available when application is minimized or not focused.

property Count: Integer read GetCount;

Number of joysticks connected in the system. If no joysticks are connected, this value will be zero.

property Initialized: Boolean read FInitialized;

Indicates whether the component has been properly initialized and is ready to be used.

property Items[Index: Integer]: TAsphyreJoystick read GetItem;

Provides access to individual joysticks connected in the system. Index can have values in range of [0..(Count - 1)]. If the specified index is out of valid range, the returned value is zero.

property WindowHandle: THandle read FWindowHandle write FWindowHandle;

The handle of the application's main window. This should be properly set before initializing the component as it will not work otherwise.
