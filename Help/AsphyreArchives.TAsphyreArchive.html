Declaration

type TAsphyreArchive = class(TObject)

Description

Asphyre archive implementation class that provides methods for reading, writing, deleting and renaming records in compressed (and possibly encrypted) archive; Zlib is used for data compression, CRC32 is used for data integrity checks and Asphyre's native 128-bit XTEA cipher is used for data encryption.

Note that this implementation is not meant for concurrent access so modifying archives from different threads and/or applications simultaneously is not supported. The data can be read by different threads/applications only if it is not being written/updated at the same time and it is the application's responsibility to ensure this.

Description

Methods

function IndexOf(const Key: UniString): Integer;

Returns index of the record that has the specified key. If no record with such key exists, the returned value is -1.

function OpenFile(const AFileName: StdString): Boolean;

Opens the specified file and refreshes the record list. If the specified file is already open, this method does nothing (in this case, Refresh can be called to explicitly refresh the record list).

function ReadMemStream(const Key: UniString; MemStream: TMemoryStream): Boolean;

Reads existing record from archive with the given key to the specified "memory" stream. The difference between this method and ReadStream is the explicit use of TMemoryStream making this method faster to execute. The entire contents of the given stream are overwritten. If the method succeeds, True is returned and the specified stream will contain the record's data block with position set to zero. If the method fails, False is returned and the stream will be empty.

function ReadRecord(const Key: UniString; out Data: Pointer; out DataSize: Integer): Boolean;

Reads existing record with the specified key from the archive and returns its raw data. The memory is allocated within this method and it's the caller responsibility to release the memory after it is no longer being used by using FreeNullMem(Data). If the method succeeds, True is returned; if the method fails, False is returned, Data is set to Nil and DataSize is set to zero.

function ReadStream(const Key: UniString; Stream: TStream): Boolean;

Reads existing record from archive with the given key to the specified stream. If the method succeeds, True is returned and the data block will be written to stream at current position. If the method fails, False is returned and the stream remains unchanged.

function RemoveRecord(const Key: UniString): Boolean;

Removes the given record from archive. The returned value is True if the method succeeds and False otherwise.

function RenameRecord(const Key, NewKey: UniString): Boolean;

Changes the name of the given record updating the record table and writing it to disk. The returned value is True if the method succeeds and False otherwise.

function WriteRecord(const Key: UniString; Source: Pointer; SourceSize: Integer; RecType: TArchiveRecordType = artFile; RecDate: TDateTime = 0.0): Boolean;

Writes a new record with the specified key and data. If a record with the same name exists, it will be removed first. If the specified date/time is zero or no date/time is specified, the current date and time will be used instead. The method returns True if the record has been written successfully or False if there was a problem.

function WriteStream(const Key: UniString; Stream: TStream; RecType: TArchiveRecordType = artFile; RecDate: TDateTime = 0.0): Boolean;

Writes a new record with the specified key from the given stream. The stream's current position is used and the remaining data is read; therefore, to use the entire stream, make sure to call Stream.Seek(0, soFromBeginning) first. If a record with the same name exists, it will be removed first. If the specified date/time is zero or no date/time is specified, the current date and time will be used instead. The method returns True if the record has been written successfully or False if there was a problem.

procedure Refresh();

Reloads the record table for the given archive, updating the list of records. This can be used to ensure that the list of records is current.

Properties

property FileName: StdString read FFileName write SetFileName;

The file name of the archive. This can be interpreted differently depending how the global variable ArchiveTypeAccess is set. Setting this property may cause the list of records to be refreshed when a different file name is being set; Ready property will also be updated as well. For a more flexible and controlled approach, consider using OpenFile method instead.

property FileSize: Integer read FFileSize;

Physical size of the archive on disk.

property OpenMode: TArchiveOpenMode read FOpenMode write FOpenMode;

How the file should be handled and accessed.

property Password: Pointer read FPassword write FPassword;

128-bit password that will be used for reading and writing encrypted records within archive. If set to Nil (by default), the data is stored unsecurely. Secure records require this password to be set, trying to read them without password will fail. The data pointed by this property should have 16 bytes or 128 bits, the encryption uses Asphyre's native 128-bit XTEA cipher.

property Ready: Boolean read FReady;

Indicates whether the archive with name set in FileName has been opened property and can be used for reading and/or writing records.

property RecordChecksum[Index: Integer]: Cardinal read GetRecordChecksum;

Returns the record's CRC32 Checksum for the given index. If the index is outside of [0..(RecordCount - 1)] range, the returned value is zero.

property RecordCount: Integer read GetRecordCount;

Number of existing records in the archive.

property RecordDate[Index: Integer]: TDateTime read GetRecordDate;

Returns the record date and time for the given index. If the index is outside of [0..(RecordCount - 1)] range, the returned value is zero.

property RecordKey[Index: Integer]: UniString read GetRecordKey;

Returns key of the record specified by the given index, which should be in range of [0..(RecordCount - 1)]. If the index is outside of valid range, empty string will be returned.

property RecordOrigSize[Index: Integer]: Cardinal read GetRecordOrigSize;

Returns the original (uncompressed) record's size for the given index. If the index is outside of [0..(RecordCount - 1)] range, the returned size is zero.

property RecordPhysSize[Index: Integer]: Cardinal read GetRecordPhysSize;

Returns the physical record's size on disk for the given index, which does not include its internal header or space used in record table. If the index is outside of [0..(RecordCount - 1)] range, the returned size is zero.

property RecordSecure[Index: Integer]: Boolean read GetRecordSecure;

Returns whether the record at the specified index is encrypted or not. If the index is outside of [0..(RecordCount - 1)] range, the returned value is False.

property RecordType[Index: Integer]: TArchiveRecordType read GetRecordType;

Returns the record type for the given index. If the index is outside of [0..(RecordCount - 1)] range, the returned type is artFile.
