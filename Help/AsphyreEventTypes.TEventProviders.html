Declaration

type TEventProviders = class(TObject)

Description

List of all available event providers based on observer pattern where subscriber classes are notified of the events sent by other classes.

Description

Methods

function Add(): TEventProvider;

Adds a new provider to the end of the list and returns the reference to its class.

function IndexOf(AProvider: TEventProvider): Integer;

Returns the index of the specified provider in the list.

function Insert(): Integer;

Inserts a new provider to the end of the list and returns its index.

procedure AddExecOrder(const AClassName: StdString);

Defines the next execution order for all event callback functions of in all existing providers for the given class. This method should be called in a sequence for all registered classes to define in which order they should receive events.

procedure Clear();

Removes all providers from the list.

procedure ClearExecOrder();

Clears all the event ordering initially specified by AddExecOrder for all classes and callback functions.

procedure MarkEventListsDirty();

Notifies all providers that their event priority list is dirty and should be updated the next time an event occurs. This can occur when the ordering list has been changed.

procedure Remove(Index: Integer);

Removes provider at the specified index from the list.

procedure Unsubscribe(const AClassName: StdString);

Unsubscribes all existing event callback functions for the specified class from all existing providers.

Properties

Number of registered provider classes in the list.

Access to individual provider classes in the list.
