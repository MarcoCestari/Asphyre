Description

Methods

function Initialize(): Boolean;

Initializes the component and prepares DirectInput interface. If the method succeeds, the returned value is True and False otherwise. WindowHandle must be properly set before for this call to succeed.

function Update(): Boolean;

Updates the state of mouse buttons and calculates the displacement that occurred since the previous call. If the method succeeds, the returned value is True and False otherwise.

procedure Finalize();

Finalizes the component releasing DirectInput interface.

Properties

property Background: Boolean read FBackground write FBackground;

Determines whether the input should still be available when application is minimized or not focused.

property BufferSize: Integer read FBufferSize write FBufferSize;

The size of mouse buffer that will store the events. It is recommended to leave this at its default value unless Update is not called often enough; a larger buffer will accomodate more mouse movement events.

property ClearOnUpdate: Boolean read FClearOnUpdate write FClearOnUpdate;

Determines whether the status of mouse buttons should be cleared each time Update is called.

property Displace: TPoint2px read FDisplace;

The mouse displacement computed since the previous call to Update.

property Exclusive: Boolean read FExclusive write FExclusive;

Determines whether the access to the mouse should be exclusive for this application.

property Initialized: Boolean read FInitialized;

Indicates whether the component has been properly initialized and is ready to be used.

property Pressed[Button: Integer]: Boolean read GetPressed;

Returns True if the specified button has been pressed since the previous call to Update. If the button has not been pressed, the returned value is False. The first button has index of zero.

property Released[Button: Integer]: Boolean read GetReleased;

Returns True if the specified button has been released since the previous call to Update. If the button has not been released, the returned value is False. The first button has index of zero.

property WindowHandle: THandle read FWindowHandle write FWindowHandle;

The handle of the application's main window. This should be properly set before initializing the component as it will not work otherwise.
