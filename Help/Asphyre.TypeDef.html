Description

Functions and Procedures

procedure FreeNullMem(var Value);

Calls FreeMem for the given value and then sets the value to Nil.

procedure PopFPUState();

Restores the FPU state stored that was previously added to the stack. If the stack is empty, this function does nothing.

procedure PushClearFPUState();

Adds the current FPU state to the stack and clears FPU state so that all FPU exceptions are disabled. If the stack is full, this function still disables FPU exceptions without saving original FPU state.

procedure PushFPUState();

Adds the current FPU state to the stack. If the stack is full, this function does nothing.

SizeFloat = Double;

General floating-point type. On 64-bit platform it is an equivalent of Double for better real-time performance, while on 32-bit systems it is an equivalent of Single.

StdString = string;

Standard string type that is compatible with most Delphi and/or FreePascal functions that is version dependant. In latest Delphi versions it is considered Unicode, while older versions and FreePascal consider it AnsiString.
