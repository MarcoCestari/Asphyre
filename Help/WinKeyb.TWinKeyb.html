Description

Methods

function CharKey(Ch: Char): Integer;

Given the specified character, this function returns its virtual key code.

Properties

property Key[vKey: Integer]: Boolean read GetKey;

Returns the status of a single keyboard key. VCode is the virtual key code usually defined by VK_[KeyName] constants.

property KeyName[vKey: Integer]: StdString read GetKeyName;

Returns the name of the given key as it is described by underlying OS. VCode is the virtual key code.

property KeyPressed[vKey: Integer]: Boolean read GetKeyPressed;

Indicates whether the key has been pressed since the last time it was checked. However, due to multitasking, a query from another application or even query to Key can reset this state so it should not be relied upon. VCode is the virtual key code.
