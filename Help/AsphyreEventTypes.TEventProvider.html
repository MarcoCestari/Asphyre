type TEventProvider = class(TObject)

Description

Event subscription class implementing observer pattern. Recipient classes are added to the subscriber list to receive notifactions of events sent by other classes.

Description

Methods

function Notify(Sender: TObject; Param: Pointer = nil): Boolean;

Send event notification to all subscribed classes and their callback functions, filtered through EventValidator event.

function SetClassPriority(const AClassName: StdString; Priority: Integer): Boolean;

Sets the secondary priority of all event callback functions registered to the specified class. This secondary priority is used after the first primary priority is applied (the one passed to Subscribe function).

function Subscribe(const AClassName: StdString; Callback: TEventCallbackType; Priority: Integer = -1): Integer;

Subscribes the given class and its event handling callback to receive all events sent from this provider. If the priority is set, the class will receive events depending on the list of priorities either after or before other events depending on priority settings. This function returns the identification number of the subscribed function, which can later be used to unsubscribe from this provider using Unsubscribe method.

procedure MarkEventListDirty();

Marks the priority list of all event callback functions dirty, so it is refreshed next time an event occurs. This should be called after changing the priority of some class.

procedure Unsubscribe(EventId: Integer);

Unsubscribes the event callback function registered using the specified ID, which is usually returned by Subscribe function.

procedure UnsubscribeClass(const AClassName: StdString);

Unsubscribes all event callbacks registered to the specified class.

The owner class that contains the list of all existing providers.
