Functions and Procedures

function cBlend(const Src, Dest: TAsphyreColor; Alpha: Integer): TAsphyreColor;

Interpolates between two given colors producing the resulting mixture. Alpha should be specified in [0..255] range.

function cClamp(const c: TAsphyreColor): TAsphyreColor;

Clamps the individual components of the given color so that the resulting color has all its components within [0..1] range.

function cColor(r, g, b, a: Single): TAsphyreColor; overload;

Constructs TAsphyreColor using individual components.

function cColor(Gray, Alpha: Single): TAsphyreColor; overload;

Constructs TAsphyreColor where red, green and blue components are set to the given grayscale value and alpha-channel specified separately.

function cColor(Gray: Single): TAsphyreColor; overload;

Constructs TAsphyreColor where red, green and blue components are set to the given grayscale value and alpha-channel set to one.

function cCubic(const c1, c2, c3, c4: TAsphyreColor; Theta: Single): TAsphyreColor;

Computes cubic interpolation between four colors for each individual components using Catmull-Rom interpolation. Theta should be specified in [0..1] range.

function cDarken(const c: TAsphyreColor; Light: Single): TAsphyreColor;

Multiplies red, green and blue channels of the given color by the specified coefficient and returns the result.

function cGrayValue(const Color: TAsphyreColor): Single;

Calculates the gray value that appear in perceptual terms from the source color.

function cLerp(const Src, Dest: TAsphyreColor; Alpha: Single): TAsphyreColor;

Interpolates between two given colors producing the resulting mixture. Alpha should be specified in [0..1] range.

function cModulateAlpha(const c: TAsphyreColor; Alpha: Single): TAsphyreColor;

Multiplies alpha-channel from the given color by the specified coefficient and returns the resulting color with red, green and blue components unchanged.

function cNegative(const c: TAsphyreColor): TAsphyreColor;

Computes negative color from the source by subtracting each individual component from one.

function cNoAlpha(const Src: TAsphyreColor): TAsphyreColor;

This function takes the source color and returns the same color but with alpha-channel set to one.

function cWrap(const c: TAsphyreColor): TAsphyreColor;

Similar to cClamp this function makes sure that all components stay within the range of [0..1]. However, if one of the components overpasses the limit, it will be "wrapped" from the limit; for example, value of 1.2 becomes 0.8, 1.1 becomes 0.9, -0.3 becomes 0.3, -1.25 becomes 0.25 and so on.
