Unit

AsphyreJoystick

Declaration

type TAsphyreJoystick = class(TObject)

Description

Joystick input class that uses DirectInput 8 for retrieving the state of the joystick, its axes buttons and other parameters. This class is usually created inside TAsphyreJoysticks for every joystick connected in the system.
