Asphyre.Textures

Declaration

type TAsphyreRenderTargetTexture = class(TAsphyreCustomTexture)

Description

Render target texture specification which supports drawing the entire scene directly on the texture. This type of texture may not be supported on some providers. It can be used for advanced multi-pass rendering effects.
