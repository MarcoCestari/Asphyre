Unit

Asphyre.Math.Sets

Declaration

type TPointList = class(TObject)

Description

List of 2D integer point holders that in addition to the vector itself contain a pointer to custom user data each. This list can be used when an association is needed between 2D points and some sort of data.

Description

Methods

function IndexOf(const APoint: TPoint2px): Integer;

Returns the first occurrence of the given 2D point in the list. If the given point is not found, this function returns -1.

function Insert(const APoint: TPoint2px; AData: Pointer = nil): Integer; overload;

Inserts the specified 2D point and custom user data to the end of list, returning the index where they were placed.

function Insert(x, y: Integer; AData: Pointer = nil): Integer; overload;

Inserts a new 2D point specified using individual coordinates along with custom user data to the end of list, returning the index where they were placed.

procedure AddFrom(const Source: TPointList);

Adds all elements from the source list to this one at the end of the list.

procedure Clear();

Removes all elements from the list.

procedure CopyFrom(const Source: TPointList);

Copies all elements from the source list to this one. The new list becomes the exact copy of the source one.

procedure Exclude(const APoint: TPoint2px);

Excludes the first occurrence of the given 2D point from the list.

procedure Include(const APoint: TPoint2px; AData: Pointer = nil);

Includes the given 2D point and custom user data to the list. This method checks if the 2D point with the given coordinates already exists and in this case, does nothing. The custom user data is not checked.

procedure Remove(Index: Integer);

Removes the element at given index, shifting all elements by one from that location and further. If the index is outside of valid range, this method does nothing.

Properties

property Count: Integer read DataCount;

The number of elements in the list.

property Item[Index: Integer]: PPointHolder read GetItem;

This property gives access to individual elements in the list by returning a reference to element at the given index, which should be in range of [0..(Count - 1)]. If the index is outside of valid range, Nil is returned.

property MemAddr: Pointer read GetMemAddr;

Returns pointer to the first element in the list or Nil otherwise.

property Point[Index: Integer]: PPoint2px read GetPoint;

This property allows accessing individual 2D points in the list directly ignoring the custom user data field.
