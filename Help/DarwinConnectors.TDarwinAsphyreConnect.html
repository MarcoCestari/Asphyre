function Init(Context: TContext3D): Boolean;

Creates the connection between Asphyre and FireMonkey on iOS, returning True if the connection is successful, and False otherwise. If the connection has previously been established, this function does nothing and returns True. This function can be called as many times as possible in timer events to make sure that the connection remains established.

Parameters
Context
Valid FireMonkey's context taken from the main form (e.g. use Self.Context in the form's code).
procedure Done();

Finalizes the connection between Asphyre and FireMonkey.
