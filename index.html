    Asphyre is a cross-platform framework for developing 2D/3D video games and interactive business applications

    Project maintained by MarcoCestari Hosted on GitHub Pages — Theme by mattgraham

    Asphyre is a framework for developing 2D/3D video games and interactive business applications. It aids the developer with resource management, displaying 2D and 3D graphics in real-time, receive keyboard, mouse and joystick input, and multiplayer capabilities.
